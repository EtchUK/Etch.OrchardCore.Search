# Etch.OrchardCore.Search

Module for [Orchard Core](https://github.com/OrchardCMS/OrchardCore) for adding site search.

## Build Status

[![Build Status](https://secure.travis-ci.org/etchuk/Etch.OrchardCore.Search.png?branch=master)](http://travis-ci.org/etchuk/Etch.OrchardCore.Search)

_This module is currently under development and will be made available on NuGet when usable._

## Orchard Core Reference

This module is referencing the beta 3 build of Orchard Core ([`1.0.0-beta3-71077`](https://www.nuget.org/packages/OrchardCore.Module.Targets/1.0.0-beta3-71077)).

## Installing

[Download the source](https://github.com/etchuk/Etch.OrchardCore.Search/archive/master.zip) or clone the repository to your local machine. Add the project to your solution that contains an Orchard Core project and add a reference to Etch.OrchardCore.Search.

Once the module is in a usable state we'll make it available via NuGet.
